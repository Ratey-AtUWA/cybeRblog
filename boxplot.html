how to boxplot

For exploratory data analysis I draw a lot of boxplots. Also, the data that I typically look at have mainly positively skewed variables. When drawing boxplots, this means that some categories can be hard to visualise because they have much lower concentrations than others, and the high values dominate the plot (see panel (a) in the attached image).

We usually deal with this by log transformation - panel (b) shows this, and the variables (and therefore boxplot boxes) now have more symmetrical distributions which are all clearly visible on the plot. But the y axis (concentration, in this case) doesn’t show the actual values.

Panel (c) shows the plot with a log-transformed axis scale, but this is not really satisfying either since the distribution is still skewed but plotted on the compressed log-transformed y-axis.

I think the best option, shown in panel (d), is to plot the log-transformed variable, but suppress the automatic axis and manually add a y-axis with a transformed scale. It’s a bit trickier, but worth it. Now we can compare all the factor levels easily, yet still have actual values on the axis (e.g. for comparison with environmental guidelines).

Four styles of boxplots based on the same data. 

with(afsed1922, boxplot(Zn ~ Zone, col=rainbow(11,s=0.35,end=0.75), 
                        las=2, xlab="", cex = 1.4, cex.lab = 1.4))
mtext("Sampling Zone",1,3,font=2)
mtext("(a)", 3, -1.2, adj = 0.03, font = 2)
mtext("Untransformed variable or axis", 3, 0.2, font=3)

with(afsed1922, boxplot(log10(Zn) ~ Zone, col=rainbow(11,s=0.35,end=0.75), 
                        las=2, xlab="", cex = 1.4, cex.lab = 1.4,
                        ylab=expression(bold(paste(log[10],"(Zn)")))))
mtext("Sampling Zone",1,3,font=2)
mtext("(b)", 3, -1.2, adj = 0.03, font = 2)
mtext("Log-transformed variable", 3, 0.2, font=3)

with(afsed1922, boxplot(Zn ~ Zone, log = "y", col=rainbow(11,s=0.35,end=0.75), 
                        las=2, xlab="", cex = 1.4, cex.lab = 1.4))
mtext("Sampling Zone",1,3,font=2)
mtext("(c)", 3, -1.2, adj = 0.03, font = 2)
mtext("Log-transformed axis", 3, 0.2, font=3)

with(afsed1922, boxplot(log10(Zn) ~ Zone, col=rainbow(11,s=0.35,end=0.75), 
                        las=2, xlab="", cex = 1.4, cex.lab = 1.4, yaxt="n",
                        ylab = "Zn"))
axis(2, at=log10(c(10,20,50,100,100,200,500,1000,2000,5000)), 
     labels=c(10,20,50,100,100,200,500,1000,2000,5000),las=2)
mtext("Sampling Zone",1,3,font=2)
mtext("(d)", 3, -1.2, adj = 0.03, font = 2, col = "blue3")
rect(-0.5,log10(10000),12.5,log10(20000), border = NA, col = "#FFFF80")
mtext("Best option: log variable, custom axis", 3, 0.2, font=3, col = "blue3")
